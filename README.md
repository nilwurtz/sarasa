# sarasa

Base64 image save project

## setup

```
$ npm i
```

## run

```
$ npm build
$ node dist/main.js ~/file/path
```

## dev

### dev run

```
$ npm run dev ~/file/path
```

### test

```
$ npm test
```

### for yanmaga

```
$ $$(".-cv-page-content canvas").filter((e) => e.getBoundingClientRect().y === 0 &&  e.getBoundingClientRect().x > 0 )[0].toDataURL("image/png")
```
